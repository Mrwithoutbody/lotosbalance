# LOTOS BALANCE — Pitch Deck v4

Statyczny pitch deck (HTML + CSS, inline SVG, bez zależności). 15 slajdów z nawigacją strzałkami / klawiaturą.

## Podgląd

- **Lokalnie:** otwórz `v4-html/index.html` w przeglądarce.
- **Online (GitHub Pages):** https://mrwithoutbody.github.io/lotosbalance/

## Struktura

```
v4-html/
├── index.html        # nawigacja (iframe + strzałki)
├── 01-cover.html … 15-contact.html
└── style.css         # Swiss design system
```

## Uwaga

Strona oznaczona `noindex, nofollow` — nie powinna być indeksowana przez wyszukiwarki.
Repo publiczne ze względu na GitHub Pages na koncie darmowym; treść (wycena, cap table,
dane kontaktowe) jest dostępna dla każdego, kto zna link.
