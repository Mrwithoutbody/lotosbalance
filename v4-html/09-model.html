lotosPayments — własna warstwa

Wszystkie transakcje przez lotosPayments — warstwę na protokole x402 (Coinbase + Cloudflare + Linux Foundation). User: BLIK / przelew, natychmiastowo. Pod spodem: settlement na Base L2, koszt transakcji ~0.01 PLN.

  • Natychmiastowy settlement (vs 14–30 dni Stripe)
  • Transparentny split payment — obie strony widzą tę samą kwotę
  • Marginalny koszt transakcji — więcej marży zostaje na platformie
  • Komponent reużywalny — docelowo poza ekosystemem LOTOS BALANCE

Unit economics (hipoteza): ARPU blended ~14–22 PLN/mies., CAC organiczny przez twórców ~5–10 PLN, blended CAC ~15–25 PLN. LTV/CAC > 5× przy retencji 9+ mies. Marża brutto > 75%.
